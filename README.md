# AI Workflows for HR Processes

This repository contains our student project for the course:

**Project 2: Designing AI Workflows to Enhance HR Processes**  
at the Chair of Information Systems and Transformation Management,  
University of Duisburg-Essen.

## Project Goal

The goal of this project is to design and prototype AI-supported workflows for HR processes.  
The project explores how workflow automation tools and AI can improve tasks such as:

- Employee onboarding
- HR request handling
- Administrative workflows
- ERP-integrated HR processes

## Technologies

The project may include technologies such as:

- n8n
- ERPNext
- AI/LLM integrations
